📋 الخطوات المطلوبة (5 دقائق):

┌─────────────────────────────────────────────────────────────┐
│ 1️⃣  افتح Cloudflare Dashboard                              │
└─────────────────────────────────────────────────────────────┘

   🔗 https://dash.cloudflare.com
   
   → اختر: Workers & Pages
   → اختر Worker الخاص بك (مثلاً: raqim-api)
   → اضغط: "Edit Code"

┌─────────────────────────────────────────────────────────────┐
│ 2️⃣  انسخ الكود الجديد                                      │
└─────────────────────────────────────────────────────────────┘

   افتح ملف: WORKER-COMPLETE-CODE.ts
   
   انسخ الكود بالكامل (Ctrl+A ثم Ctrl+C)
   
   الصقه في Cloudflare Editor (استبدل الكود القديم)

┌─────────────────────────────────────────────────────────────┐
│ 3️⃣  احذف GEMINI_API_KEY                                    │
└─────────────────────────────────────────────────────────────┘

   في نفس Worker:
   
   Settings → Variables → Environment Variables
   
   ابحث عن: GEMINI_API_KEY
   
   اضغط: "Edit" → "Delete"

┌─────────────────────────────────────────────────────────────┐
│ 4️⃣  تأكد من وجود OpenAI Key                               │
└─────────────────────────────────────────────────────────────┘

   تحقق من وجود المتغيرات التالية:
   
   ✅ OPENAI_API_KEY = sk-proj-...
   ✅ OPENAI_OSS_MODEL = gpt-4o-mini
   ✅ ALLOWED_ORIGIN = *
   
   إذا لم يكن موجوداً، أضفه:
   → "Add variable" → اسم المتغير + القيمة

┌─────────────────────────────────────────────────────────────┐
│ 5️⃣  احفظ ونشر                                             │
└─────────────────────────────────────────────────────────────┘

   اضغط: "Save and Deploy"
   
   انتظر 10-20 ثانية للنشر

┌─────────────────────────────────────────────────────────────┐
│ ✅ اختبار                                                  │
└─────────────────────────────────────────────────────────────┘

   افتح موقعك: https://raqim2026-ai.pages.dev
   
   جرب أي أداة (مثلاً: كاشف النصوص)
   
   يجب أن تعمل بدون أخطاء!

❓ مشاكل شائعة:

Problem: "Missing secret OPENAI_API_KEY"
Solution: أضف المتغير في Settings → Variables

Problem: الأدوات لا تعمل
Solution: تأكد من نسخ الكود الجديد كاملاً

Problem: CORS errors
Solution: تأكد من ALLOWED_ORIGIN = *
